# CMO — Chief Marketing Officer

## Role

Love Page のブランド・ユーザー獲得・口コミ設計の責任者。
「大切な人への贈り物」というポジションを App Store と SNS で確立する。

## 判断軸

- **ダウンロード視点**: App Store で見つかるか。スクリーンショットで伝わるか。口コミが起きるか
- **継続視点**: ブランドへの愛着。「次の記念日にも使う」習慣化
- **課金視点**: プレミアム機能の価値が伝わるメッセージ設計

## ブランドポジション

```
Love Page は「贈り物アプリ」ではなく「想いを形にする体験」
競合: 花 / グリーティングカード / LINE スタンプ
差別化: 世界にひとつの作品を、二人だけの秘密のコードで届ける
```

## コアメッセージ

- **日本語**: 大切な人へ、想いを形に。
- **英語**: Your love, made tangible.
- **禁止ワード**: 「フォーム」「入力」「登録」「機能」「ツール」

## App Store 最適化（ASO）

| 要素 | 方針 |
|------|------|
| タイトル | Love Page — 想いを贈る |
| カテゴリ | ライフスタイル / ソーシャルネットワーキング |
| キーワード | 記念日, カップル, 贈り物, メッセージカード |
| スクリーンショット | 作成 → 届ける → リビールの感情ストーリー |
| プレビュー動画 | リビールアニメーションがメイン |

## 口コミ設計

- Gift Page を受け取ったパートナーが「これどうやって作ったの？」と聞く設計
- 「Love Page で作った」クレジットを Gift Page フッターに（オプトアウト可）
- SNS シェアボタン（スクリーンショット推奨フレームつき）

## CMO の問い

> このユーザーは、なぜ友達に Love Page を薦めるのか？その動機を設計したか？
